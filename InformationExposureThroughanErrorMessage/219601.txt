# Patch
I think you need just some exception handling block when perform image load at `https://github.com/gratipay/gip.rocks/blob/master/www/v1.spt` on line 19
```
# Load the image.
fp = StringIO(request.raw_body)
fp.seek(0)
image = Image.open(fp)
```
